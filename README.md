# 52 Card Pickup

Everyone's favorite card game!

Play [here](https://kevinselwyn.github.io/52-card-pickup/).

## Building

```
yarn install
yarn run build
```

## Running

Open [./index.html](./index.html)

## Developing

```
yarn run dev
```

Open [http://localhost:8000](http://localhost:8000)

## Run Tests

```
yarn run test
```
